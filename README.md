# DentoBot--Your-smart-dental-assistance
UI/UX research and design project "DentoBot", a smart dental awareness chatbot focused on improving oral health education and patient engagement.
# Project Overview
The purpose of this project is to design a user-centered dental awareness chatbot that helps patients access clear and reliable oral health information while reducing repetitive communication for dental professionals. Through UX research and design methods, this project explores how a conversational interface like Dentobot can improve patient education, engagement, and overall user experience in a healthcare setting.
